# architrathi.ca

Personal portfolio website showcasing Shopify engineering work, case studies, and independent product projects.

## Overview

This site serves as a curated showcase of:

- Shopify-focused client case studies
- Performance-driven storefront engineering
- Accessibility and compliance implementations
- Independent React and product experiments
- Interactive UI work and animation systems

Built to reflect both technical depth and product-level design thinking.

---

## Tech Stack

- **Next.js 14**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion / Motion**
- **Vercel Deployment**

---

## Features

- Custom animated case study slider
- Dynamic modal case study architecture
- Conditional case study rendering
- Responsive project grid with curated repositories
- Performance-optimized image handling
- Fully responsive design system

---

## Project Structure

/src

Case studies are modularized per client and dynamically injected based on slug.
